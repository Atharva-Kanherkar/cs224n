# cs224n

Working through Stanford CS224N (NLP with Deep Learning) concepts by hand,
one exercise at a time. Each numbered folder is a self-contained exercise:
a scaffold with stub functions I have to implement, a test suite that has to
pass, and a README that explains the concept and the task without handing
over the implementation.

## Exercises

1. [`01-skipgram-negative-sampling/`](01-skipgram-negative-sampling/) — Skip-Gram
   word2vec with negative sampling: build the training pipeline by hand, from
   vocabulary building through sparse gradient updates on the embedding matrix.
